Menu/Settings/InGame UI System

- Tweening components(have different tween types available)
- Particles In the UI
- audio for selections
- Fade in tool tip with more details about the option
- shadow on edges of viewports for scrolling
- thin scroll bar
- modal window?

Each tab button opens a panel to the right
Each panel can be broken up into sections
Each section will have a heading, option list and interaction type(dropdown list, slider, checkbox, enable/disable, popup window)
and a seperation line at the bottom of the section

Instead of a section syustem an inventory grid with drag and drop menu can be used deatiled out below
Inventory/Skills System

- weapons, equipment, consumables, quests 
- drag and drop
